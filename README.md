# zwy-api
使用方法
本地配置一个域名指向该目录

# 接口目录
### 日志系统-中台：pmp-mf.yaml
### 日志系统-前台：pmp-front.yaml
### 商城小程序-中台：store-mf.yaml

# Example
### 例如：本地配置一个服务器：http://api.zhaowy.com
### 日志系统-中台：http://api.zhaowy.com/pmp-mf.yaml
### 日志系统-前台：http://api.zhaowy.com/pmp-front.yaml
### 商城小程序-中台：http://api.zhaowy.com/store-mf.yaml
